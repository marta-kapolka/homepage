Tydzień IV

- CSS -

  • overflow: visible (hidden, scroll, auto) - określa zachowanie elementu jeśli treść wychodzi poza opakowanie

- CSS -

box model

  • padding - należy do elementu
  • margin - znajduje się poza elementem
  • margin: auto - element musi mieć zdefiniowaną szerokość!
  • box-sizing: border-box - wymiary są definiowane dla całego elementu - łącznie z padding i border, content jest wynikowy

- CSS -

position

  • static - domyślna
  • relative
  • absolute - pozycjonowanie względem rodzica (rodzic nie może mieć position static)
  • fixed - pozycjonowanie względem okna przeglądarki
  • sticky - jak fixed, ale tylko w obrębie rodzica, rodzic nie może mieć overflow, element rezerwuje miejsce na stronie

- CSS -

float

  • nie służy do tworzenia layoutów
  • nadawany elementowi, który ma być opływany przez inny element (obrazek opływany przez tekst)
  • clear: both (left, right) - własność nadawana pierwszemu elementowi, który ma nie opływać elementu, powoduje rozciągnięcie opakowania opływanego elementu, bo tak!

- GIT -

Komendy w konsoli

  • got init - utworzenie repozytorium
  • git status - sprawdzenie stanu repozytorium
  • git add . - dodanie plików do planowanego commita
  • git commit -m "opis zmian" - commit
  • gitk - cała historia repozytorium
  • git push - wypchnięcie zmian do zewnętrznego repozytorium

- JS -

Typy danych

  • string - tekst
  • number - liczba
  • bigint - duża liczba (> 253)
  • boolean - true / false
  • null - celowo niezdefiniowana wartość
  • undefined - brak definicji
  • * symbol - tworzy "unikalne opakowanie"

- DOM -

Document Object Model

  • Strona "złożona" przez przeglądarkę na podstawie "instrukcji" (HTML, modyfikacje wykonane przez JS, elementy typu zdjęcia, fonty itd.)
  • JS komunikuje się ze stroną poprzez DOM

- DOM / JS -

querySelector

  • document.querySelector
    ('.selector')
  • querySelector zwraca cały element wraz z dziećmi (node - węzeł)
  • '.selector' - tak samo jak w CSS
  • klasy do JS nazywamy block__element--js
  • * nie korzystamy już z getElementBy...

- DOM / JS -

zmienna.innerHTML

  • zmienna.innerHTML - zwraca zawartość elementu pod zmienną
  • zmienna.innerHTML = 'treść' - nadpisuje treść elementu pod zmienną
  • za pomocą innerHTML da się "wklejać" całe elementy HTML

- BEM -

Block | Element | Modifier

  • konwencja nazewnictwa klas w CSS
  • jeśli element posiada modyfikator to dodajemy dwie klasy: class="hand__finger hand__finger--thumb"

- HTML -

  • div - tylko jako opakowanie do stylowania, gdy nie istnieje odpowiedni znacznik semantyczny

- Jednostki -

  • % - liczone względem rodzica
  • vw, vh - jak procenty, ale liczone względem okna przeglądarki
