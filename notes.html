Tydzień II

- VSC -

  • Magiczny zestaw przycisków w VSC: !+tab - kawałek HTMLa z głowy
  • Formatowanie dokumentu HTML - shift+alt+F

- HTML -

  • Semantyka, semantyka i jeszcze raz semantyka!
  • Strukturę HTMLa tworzymy myśląc o treści, dopiero potem kombinujemy ze stylowaniem.

Tydzień III

- CSS -

  • Stylowanie po klasach!
  • Najlepiej żeby element miał jedną klasę.
  • Specyficzność selektorów:
    1. !important
    2. inline
    3. #id
    4. .class
    5. element

- Kanban -

Jak planować pracę?

  • Zwykle 3 kolumny:
    TO DO, DOING, DONE
  • W kolumnie DOING powinna się znajdować tylko jedna rzecz!
  • Warto zdefiniować warunki wykonania zadania

- Internety -

Z czego składa się adres strony?

  • http:// - protokół
  • cotenfrontend.pl - host (adres serwera)
  • :443 - port
  • / - root (!!!)
  • index.html - ścieżka dostępu do zasobu
  • ?parametr=wartość - parametry

- Internety -

Co się dzieje w przeglądarce?

  • Wpisuję adres
  • Przeglądarka przerabia z www... na IP serwera (DNS)
  • Przeglądarka odpytuje serwer o index
  • Serwer wysyła HTMLa
  • Przeglądarka wczytuje HTMLa i szuka pozostałych zasobów
  • Przeglądarka renderuje stronę

- Internety -

Kody HTTP

  • 1xx - czekaj
  • 2xx - wszystko OK
  • 3xx - szukasz w złym miejscu, ale przekieruję bo wiem gdzie
  • 4xx - ni mo :(
  • 5xx - błąd serwera

- Terminal -

Komendy w Windowsie

  • cd C:\ - przejście do C:
  • dir - zawartość katalogu
  • cd .. - powrót do katalogu wyżej
  • cd nazwa katalogu - przejście do danego katalogu
  • code . - uruchomienie VSC w danym katalogu
  • cd - wyświetlenie aktualnej ścieżki
  • mkdir nazwa katalogu - utworzenie katalogu o danej nazwie

Tydzień IV

- GIT -

Komendy w konsoli

  • got init - utworzenie repozytorium
  • git status - sprawdzenie stanu repozytorium
  • git add . - dodanie plików do planowanego commita
  • git commit -m "opis zmian" - commit
  • gitk - cała historia repozytorium
  • git push - wypchnięcie zmian do zewnętrznego repozytorium
